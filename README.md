# Batalha_Naval.c

Jogo de Batalha Naval em C.

Modo demonstrativo onde o jogador conhece um pouco sobre a forma de operar do jogo Batalha Naval.

Navios são representados pelos números "1" e "2".

Os números "0" no formato de tabuleiro serve para representar a água do mar onde os navios estão navegando.

O objetivo final é atingir os navios em suas respectivas casas do tabuleiro.
